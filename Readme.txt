This game takes system/ command line arguments 
0- human
1- Random 
2- Minimax
3- Alpha beta

To play the game, run the play.py followed by the corresponding number of the player/ AI you want to choose.
This only take an integer input so string values wont work. 
for example in python3
python3 play.py 3 2 
this will run the game with alpha beta as player 1 and minimax as player 2
